시리즈온

콘텐츠 탐색 경험과 구매 경험 개선하기

평소 다양한 OTT 서비스를 이용하던 저는 서비스마다 각기 다른 UX 전략을 취한다는 점에서 흥미를 느꼈습니다. 그중에서도 사용자가 비교적 적었던 네이버 시리즈온을 분석해보았습니다. 과거 시리즈온을 즐겨 사용했던 경험을 바탕으로, 다른 OTT 서비스들과의 UX 차이를 비교하고 시리즈온에 적합한 전략을 고민해보고자 했습니다.

배경

치열한 OTT 경쟁 속, 시리즈온의 위치는?

코로나19 이후, OTT 시장은 자체 제작 콘텐츠와 독점 중계를 앞세운 경쟁이 치열하게 전개되었습니다. 시리즈온은 그 속에서 사용자 점유율이 낮은 편이었고, 결국 서비스는 종료되었습니다. 만약 시리즈온 서비스가 지속되었다면, 어떤 UX 전략을 통해 사용자 경험을 개선하고 수익성을 확보할 수 있었을지에 대한 가능성을 탐색해보았습니다. 또한, 시리즈온의 장점을 살린 UX 개선을 통해 더 나은 콘텐츠 경험을 제공하고자 했습니다.

설문조사

사용자들이 꼽은 시리즈온의 장점은 무엇일까요?

시리즈온 사용자들을 대상으로 한 설문 조사 결과, 아래와 같은 세 가지 강점이 특히 돋보였습니다.

1. 희귀한 콘텐츠 보유 (51%)
사용자들은 시리즈온에서만 볼 수 있는 특정 장르, 고전 영화, 독립 영화 등 희소성 높은 콘텐츠에 높은 만족도를 보였습니다. 이는 다른 OTT 플랫폼에서 쉽게 찾기 어려운 콘텐츠를 찾는 사용자들에게 큰 매력으로 작용했습니다.

2. 개별 구매의 효율성 (39%)
정기 구독 없이, 원하는 콘텐츠만 골라서 구매할 수 있는 방식이 효율적이라는 의견도 많았습니다. 자주 시청하지 않더라도 가끔씩 특정 콘텐츠만 즐기고 싶은 사용자에게 특히 적합한 구조였습니다.

3. 무료 영화 제공 (32%)
무료로 제공되는 콘텐츠는 라이트 유저들에게 접근 장벽을 낮추고, 플랫폼 체험 기회를 제공하는 요소로 작용했습니다.

서비스 분석

시리즈온과 다양한 OTT 서비스를 분석해보았습니다.

OTT 서비스는 각기 다른 전략을 취하고 있지만, 결제 방식에 따라 사용자 경험에 뚜렷한 차이가 나타납니다. 이에 따라 분석은 두 가지 유형으로 나누어 진행했습니다.

첫째는, 구독만으로 모든 콘텐츠를 자유롭게 이용할 수 있는 구독형 서비스,

둘째는, 구독 콘텐츠 외에도 개별 결제가 필요한 콘텐츠가 함께 제공되는 구독·결제 혼합형 서비스입니다.

이번 분석에서는 두 유형의 서비스가 콘텐츠를 어떻게 제공하고, 또 어떤 방식으로 차별화된 사용자 경험을 설계하고 있는지에 중점을 두었습니다.

1. 구독형 서비스: 넷플릭스, 디즈니 플러스

구독형 OTT 서비스는 정액 요금제 기반으로, 이용자는 별도의 추가 결제 없이 대부분의 콘텐츠를 자유롭게 시청할 수 있습니다. 대표적인 서비스로는 넷플릭스와 디즈니플러스가 있으며, 이들은 공통적으로 개인화된 콘텐츠 탐색 경험과 독점 콘텐츠 제공 전략을 통해 강력한 사용자 몰입을 유도하고 있습니다.

두 서비스 모두 강력한 개인화 추천 알고리즘을 바탕으로, 사용자의 시청 이력과 선호도를 분석하여 적합한 콘텐츠를 빠르게 탐색할 수 있도록 돕고 있습니다. 또한, 다양한 연관 검색 결과를 제공해 사용자가 원하는 콘텐츠에 자연스럽게 도달할 수 있는 검색 흐름을 설계했습니다.

리텐션 전략 측면에서는, 오리지널 시리즈 및 자체 제작 콘텐츠 확보를 통해 플랫폼만의 차별화를 강화하고 있습니다. 특히, 디즈니플러스는 마블·픽사·스타워즈 등 브랜드 중심의 IP 콘텐츠를 강점으로 삼고 있으며, 넷플릭스는 시즌 연계 콘텐츠와 빠른 업데이트 주기로 이용자 충성도를 높이고 있습니다.

2. 구독·결제 혼합형 서비스: 웨이브, 티빙, 왓챠, 쿠팡플레이

웨이브, 티빙, 왓챠, 쿠팡플레이와 같은 국내 OTT 서비스는 기본적인 구독 모델 위에 개별 결제 콘텐츠를 함께 제공하는 혼합형 구조를 취하고 있습니다.

이들 서비스는 공통적으로 다양한 주제와 큐레이션 기반의 콘텐츠 탐색 시스템을 제공하고 있습니다. 특정 장르, 테마, 트렌드에 맞춘 큐레이션으로 사용자가 새로운 콘텐츠를 손쉽게 발견할 수 있도록 돕고 있습니다. 또한, 검색 기능과 필터링 기능이 비교적 우수하여 사용자가 원하는 콘텐츠를 빠르게 찾을 수 있도록 설계된 것이 특징입니다.

리텐션 전략 측면에서는, 각 플랫폼이 독점 계약을 통해 확보한 콘텐츠를 중심으로 차별화를 시도하고 있습니다. 예를 들어, 티빙은 오리지널 예능 콘텐츠, 왓챠는 감성 중심의 독립 영화와 취향 기반 추천, 쿠팡플레이는 스포츠 중계와 오리지널 드라마로 고유의 색을 만들어가고 있습니다.

이처럼 구독·결제 혼합형 서비스는 콘텐츠 큐레이션, 정교한 검색 경험, 그리고 독점 콘텐츠 확보를 통해 사용자에게 콘텐츠 탐색과 소비의 경계를 매끄럽게 연결합니다.

3. 구독·결제 혼합형 서비스: 시리즈온

시리즈온은 구독과 개별 결제가 혼합된 구조를 기반으로 하는 서비스로, 웨이브·티빙·왓챠 등 국내 주요 OTT 서비스와 유사한 포지셔닝을 가지고 있습니다. 하지만 지금까지는 인기 콘텐츠 중심의 추천, 기본적인 장르 필터만 제공되는 탐색 구조, 그리고 재방문을 유도할 요소가 부족한 리텐션 설계로 인해 경쟁 서비스에 비해 사용자 경험이 제한적이었습니다.

이를 보완하기 위해, 시리즈온은 다음과 같은 방향으로 UX 전략을 발전시킬 수 있습니다.

  • 취향 기반 추천 알고리즘 도입을 통해, 사용자 개인의 시청 이력과 선호도에 따라 콘텐츠를 더욱 정교하게 추천
  • 장르, 배우, 국가 등 다양한 필터 옵션을 제공하여 사용자가 원하는 콘텐츠를 더 쉽고 빠르게 탐색할 수 있는 환경 조성
  • 무료 콘텐츠의 정기 업데이트 및 알림 기능 강화를 통해 사용자의 자발적인 재방문을 유도하고, 콘텐츠 소비를 활성화

이러한 전략을 통해 시리즈온은 추천과 탐색의 효율을 높이고, 무료 콘텐츠를 리텐션 유도 장치로 적극 활용함으로써, 혼합형 OTT 시장 내에서 차별화된 사용자 경험을 제공할 수 있는 기반을 마련할 수 있습니다.

가설

사용자들은 시리즈온에서 원하는 콘텐츠를 쉽게 찾고, 구매할 수 있을까요?

시리즈온은 사용자들이 꼽은 장점 중 하나가 '개별 구매의 효율성'이라는 점에서, 콘텐츠 탐색 후 구매까지의 흐름이 특히 중요한 서비스입니다. 하지만 경쟁 OTT 서비스들과의 비교 분석을 통해, 시리즈온은 탐색 기능이 상대적으로 약하고, 추천 및 필터링 시스템이 제한적이라는 문제점을 발견할 수 있었습니다.

이에 따라 다음과 같은 가설을 설정하고, 사용자 테스트(User Test)와 인터뷰를 진행하였습니다:

"사용자들은 시리즈온에서 원하는 콘텐츠를 찾고, 구매하는 데에 어려움을 느낄 것이다."

이 가설은 시리즈온의 실제 사용 환경에서 탐색과 구매 과정이 사용자 기대에 얼마나 부합하는지, 그리고 전환 흐름이 자연스럽게 연결되고 있는지를 확인하기 위해 설정된 것입니다.

문제점

유저 테스트로 문제를 발견했습니다.

앞서 세운 가설을 검증하기 위해, 평소 OTT 서비스를 2개 이상 이용하는 사용자 4인을 대상으로 시리즈온 앱 내에서 탐색 및 구매와 관련된 미션을 수행하도록 요청했습니다. 참여자들에게는 이용 과정에서 느낀 점을 자유롭게 말하도록 유도하여, 실제 사용 중 겪는 불편과 혼란의 지점을 파악하고자 했습니다.

Task 1. 탐색

앱을 자유롭게 탐색하며 시청할 콘텐츠를 찾아보세요.

사용자들은 시리즈온에서 콘텐츠의 분류가 직관적이지 않고, 흥미로운 콘텐츠를 찾기 어려운 탐색 환경에 불편을 느꼈습니다. 콘텐츠가 많지만 무엇을 봐야 할지 알기 어렵고, 영화 외의 콘텐츠는 잘 드러나지 않으며, 세부적인 기준(장르, 배우 등)으로 탐색하는 기능이 부족하다는 의견이 나타났습니다. 이는 시리즈온의 강점인 개별 구매 시스템을 제대로 활용하지 못하게 만들고, 탐색에서 구매까지의 전환 흐름을 방해하는 핵심 문제로 드러났습니다.

Task 2. 구매

관심이 가는 콘텐츠를 선택하고 구매를 진행해보세요.

사용자들은 콘텐츠 구매 과정에서도 불편함을 호소했습니다. 상세 페이지에서 가격 정보가 눈에 잘 띄지 않고, 구매 버튼이 화면 하단에 작게 배치되어 있어 사용자가 쉽게 발견하지 못하는 경우가 있었습니다. 또한 구매 옵션(시즌별 구매, 에피소드별 구매 등)이 명확하게 구분되지 않아 사용자들이 원하는 방식으로 콘텐츠를 선택하는 데 혼란을 겪었습니다. 특히 컨텐츠 간 가격 차이의 이유나 할인 정보가 직관적으로 제공되지 않아 구매 결정을 망설이게 만드는 요소로 작용했습니다.

Task 3. 시청 및 재구매

콘텐츠를 시청한 후, 다른 콘텐츠도 구매해보세요.

콘텐츠를 시청하고 나서 또 다른 콘텐츠를 구매하는 과정에서도 몇 가지 문제점이 발견되었습니다. 사용자들은 시청 이력과 연계된 콘텐츠 추천이 부족하다고 느꼈으며, 이전에 시청했던 장르나 배우의 다른 작품을 찾기 위해서는 다시 처음부터 검색해야 했습니다. 또한, 구매했던 콘텐츠와 관련된 시리즈나 시즌 정보가 명확하게 제공되지 않아 연속적인 콘텐츠 소비 경험이 단절되는 상황도 발생했습니다. 이는 결국 사용자가 추가 구매를 고려할 때 결정에 어려움을 주고, 콘텐츠 추천의 개인화 부재로 인해 재방문과 지속적인 이용을 저해하는 요소로 작용했습니다.

UX 전략

복잡했던 탐색 경험을, 자연스러운 구매 흐름으로

시리즈온은 콘텐츠는 많지만, 무엇을 봐야 할지 알기 어려운 탐색 구조와 중복된 검색 결과, 비효율적인 구매 흐름으로 인해 사용자가 쉽게 이탈할 수 있는 환경이었습니다.

이에 따라, 사용자가 원하는 콘텐츠를 쉽게 찾고, 구매까지 자연스럽게 이어지는 흐름을 만드는 것을 목표로 개선 방향을 설정했습니다:

  • 콘텐츠 정렬 방식을 개선해서 탐색을 더 간단하게
  • 중복 제거 및 가독성 향상
  • 상세 정보 확인부터 구매까지 매끄럽게 이어지는 UX 설계

이러한 변화는 시리즈온이 가진 '개별 구매'라는 강점을 사용자가 체감할 수 있는 경험으로 연결해주는 핵심 전략입니다.

디자인

개선 방향을 적용하여 화면을 디자인했습니다.

1. 쉬운 탐색을 돕는 구조

홈 화면

통합된 탐색 경험을 제공하는 홈 화면

기존 시리즈온의 메인 화면은 장르별로 분리된 진입 구조로 인해, 전체 콘텐츠 흐름을 한눈에 파악하기 어렵고 탐색 동선이 단절된다는 문제가 있었습니다. 이를 해결하기 위해 다음과 같은 개선을 했습니다.

[1] 영화, 방송 등 모든 장르를 하나의 화면에서 탐색할 수 있는 통합형 메인 구조로 개편

[2] 사용자는 상단 탭을 통해 원하는 콘텐츠 유형으로 빠르게 이동

[3][4] '맞춤 추천', '무료 영화', '희귀 콘텐츠' 등 시리즈온의 강점 콘텐츠가 홈 화면에 자연스럽게 노출

이 구조는 사용자가 앱을 열었을 때 무엇을 봐야 할지 고민하지 않아도 되는 탐색 경험을 제공하는 것을 목표로 설계되었습니다.

추천 방식 개선

찾지 말고 발견하세요

기존 시리즈온은 콘텐츠를 주로 인기순·최신순 중심으로 소개하여, 사용자가 자신의 취향에 맞는 콘텐츠를 발견하기 어려운 구조였습니다. 이는 흥미로운 콘텐츠를 찾기 어려워지고, 탐색이 단조롭고 수동적으로 느껴지는 문제로 이어졌습니다.

이에 따라, 다양한 기준에 기반한 콘텐츠 큐레이션 기능을 강화했습니다.

  • '맞춤 추천', '친구 기반 추천 컬렉션' 등 사용자 취향이나 맥락에 따라 탐색할 수 있도록 구조를 확장
  • '비슷한 취향의 유저가 본 콘텐츠', '테마별 추천 리스트' 등 발견의 재미를 느낄 수 있는 큐레이션 요소를 배치

이러한 개선은 사용자가 콘텐츠를 '찾는' 것이 아니라, '발견하게 만드는' 경험을 목표로 설계되었습니다.

추천 페이지

스크롤만으로 추천 콘텐츠를 쉽게 탐색

사용자가 콘텐츠를 선택할 때, 더 이상 복잡한 탐색이나 검색 없이 개인 맞춤형 추천 콘텐츠를 스크롤하며 가볍게 탐색할 수 있도록 설계했습니다.

  • 영상 썸네일 기반의 추천 피드로 콘텐츠를 직관적으로 탐색
  • 리뷰 버튼을 통해 스포일러 차단 기능과 함께 신뢰도 높은 사용자 후기 확인
  • 콘텐츠 상세 화면에서 이용 조건, 가격, 감독/출연진 등 핵심 정보를 한눈에 파악
  • 원할 경우, 즉시 대여·구매로 이어지는 직관적인 전환 버튼 배치

이 흐름은 사용자가 탐색부터 구매까지 끊김 없이 이어질 수 있도록 설계된 구조이며, 시리즈온의 '개별 구매 모델'에 최적화된 사용자 경험을 제공합니다.

2. 직관적인 검색 환경

카테고리 검색

정리된 분류와 쉬운 필터링

기존 시리즈온의 카테고리 화면은 영화와 방송이 분리되고, 검색 결과가 지나치게 많아 원하는 콘텐츠를 찾기 어려운 구조였습니다. 사용자는 장르만으로 콘텐츠를 좁혀가야 했고, 콘텐츠가 너무 많아 선택 피로도가 높았습니다.

이에 따라 개선안에서는 모든 콘텐츠를 한눈에 탐색할 수 있도록 분류 체계를 통합하고, 다양한 태그 필터 기능을 함께 적용해 사용자 경험을 개선했습니다.

  • 카테고리 메인 화면에서 TOP 100, 할인 콘텐츠, 멤버십 전용, 공포/액션 등 다양한 주제별 큐레이션을 분류
  • 콘텐츠 리스트에는 '장르', '유료/무료', '연령', '멤버십 여부' 등 상세 필터 기능을 추가하여 사용자가 원하는 조건에 맞는 콘텐츠를 쉽고 빠르게 탐색할 수 있도록 구성

이 설계는 복잡했던 탐색 과정을 단순화해, 사용자가 처음부터 명확한 기준으로 콘텐츠를 고를 수 있는 환경을 만드는 데 초점을 두었습니다.

키워드 검색

더 빠르고 정확하게 원하는 콘텐츠 찾기

기존 시리즈온의 검색 결과 화면은 동일한 콘텐츠가 중복 노출되어, 사용자가 원하는 정보를 빠르게 찾기 어려웠고, 가독성과 탐색 효율도 낮은 구조였습니다.

이에 따라, 검색 결과를 콘텐츠 기준으로 통합 정리하고 중복 항목을 제거해 핵심 정보만 명확히 보여주는 구조로 개선했습니다.

  • 하나의 콘텐츠에 대한 다양한 옵션(소장/대여/정보)을 한 카드에 통합
  • 최근 검색어, 탭별 필터 기능을 도입해 검색 흐름을 단순화

이 개선을 통해 사용자는 더 적은 스크롤과 클릭으로 원하는 콘텐츠에 도달할 수 있는 경험을 얻게 됩니다.

3. 탐색부터 구매까지 이어지는 자연스러운 흐름

상세 화면

가격과 조건을 한눈에 비교하고 더 쉽게 구매하기

기존 시리즈온은 대여·구매·시리즈 구매 등 다양한 상품 옵션이 분산되어 있어, 사용자가 각각의 조건을 직접 확인하고 비교해야 했으며, 그 과정이 복잡하게 느껴졌습니다.

이에 따라, 모든 가격과 이용 조건을 한눈에 비교할 수 있도록 상품 정보 구조를 재구성했습니다.

  • 작품 상세 화면에서 대여·구매 가격, 멤버십 혜택, 시리즈 묶음 할인을 직관적으로 배치
  • 결제 화면에서는 보유 캐시, 부족한 금액, 충전 금액까지 명확히 표시

이 설계는 사용자가 원하는 옵션을 쉽게 비교하고 빠르게 결정할 수 있도록 흐름을 단순화하며, 구매 피로도를 줄이고 합리적인 선택을 돕는 UX 경험을 제공합니다.

사용자 만족도 테스트

개선안 적용 후, 실제 사용자 피드백은 어땠을까요?

개선된 화면과 기능이 사용자 경험에 어떤 영향을 주는지 확인하기 위해, 4명의 사용자에게 Before/After 화면을 기반으로 솔직한 의견을 수집했습니다.

사용자들은 기존 UI에 대해 콘텐츠 분류가 모호하고, 검색 결과가 중복되며, 가격 비교 과정이 복잡하다는 공통적인 불편을 느꼈습니다.

반면, 개선된 UI에서는 탐색과 구매 흐름의 자연스러움, 정보 구조의 명확성을 긍정적으로 평가했습니다.

그 결과, Before 대비 평균 만족도가 전 구간에서 1.5~2점 이상 상승하는 긍정적인 피드백을 얻을 수 있었습니다.
